## 📦 مصدر البيانات — `quiz_questions.json`

كل الأسئلة في ملف JSON واحد. شكل كل سؤال:

```json
{
  "id": 1,
  "section": "أوليات في الكتاب المقدس",
  "question": "ما هو أول وعد من الله للبشر ؟",
  "type": "complete",
  "options": [],
  "answer": "أول وعد من الله للبشر كان أن نسل المرأة يسحق رأس الحية (تك ٣: ١٥)"
}
```

### تعريف الحقول

| الحقل | النوع | القيم الممكنة | الوصف |
|---|---|---|---|
| `id` | `number` | 1 – 325 | معرف فريد لكل سؤال |
| `section` | `string` | 14 قيمة — انظر جدول الأقسام | القسم الموضوعي |
| `question` | `string` | — | نص السؤال |
| `type` | `string` | `"complete"` \| `"true_false"` | نوع السؤال |
| `options` | `array` | `[]` أو `["صح", "خطأ"]` | خيارات الإجابة — فارغ للأسئلة المفتوحة |
| `answer` | `string` | — | الإجابة الكاملة مع المرجع الكتابي |

### فلترة الأسئلة حسب القسم

```js
const questions = await fetch('/questions.json').then(r => r.json());

// الأسئلة المختلطة — كل الأقسام
const all = questions;

// قسم معين
const section = questions.filter(q => q.section === 'الروح القدس');

// أسئلة عشوائية
const shuffled = questions.sort(() => Math.random() - 0.5);
```

## 🎮 منطق المسابقة

### نوعا الأسئلة

**1. `complete` — سؤال مفتوح (الغالبية)**
السؤال يظهر، والمستخدم يختار إما:
- **"عرض الإجابة"** ← يكشف الإجابة مباشرة
- **"عرض الاختيارات"** ← يولّد 3 إجابات وهمية من أسئلة أخرى + الإجابة الصحيحة

### توليد الاختيارات الوهمية (MCQ)

```js
function generateOptions(currentQuestion, allQuestions) {
  const wrong = allQuestions
    .filter(q => q.id !== currentQuestion.id)
    .sort(() => Math.random() - 0.5)
    .slice(0, 3)
    .map(q => q.answer);

  return [...wrong, currentQuestion.answer]
    .sort(() => Math.random() - 0.5);
}
```

## ➕ إضافة أسئلة جديدة

لإضافة أسئلة لـ `questions.json`:

```json
{
  "id": 326,
  "section": "أوليات في الكتاب المقدس",
  "question": "نص السؤال الجديد؟",
  "type": "complete",
  "options": [],
  "answer": "الإجابة الكاملة مع الشاهد الكتابي"
}
```

**قواعد:**
- `id` لازم يكون فريد ومتسلسل
- `section` لازم يكون من الـ 14 قيمة الموجودة، أو قسم جديد بنفس الشكل
- `type` إما `complete` أو `true_false` فقط
- لو `type` هو `true_false`، حط `"options": ["صح", "خطأ"]`
